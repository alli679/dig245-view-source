DIG 245 View Source

How to view the source code of Safari:

  1. Go to the menu bar at the top of the screen.
  2. If there is no "Develop" option, click on "Safari" instead, then select "Preferences", otherwise, skip to step 4.
  3. In the Preferences menu, select "Advanced", then check the box that says "Show Develop menu in menu bar".
  4. Select the "Develop" option on the menu bar.
  5. Select "View Page Source" on the Develop menu. You should now be able to view the webpage's source code.

3 other things you can do by using Dev Tools (Or: "3 other things you can do by joining a website's run crew"):

  • Edit code on a webpage and immediately see the changes.
  • Copy code that you want to use for your own projects.
  • Quickly find an element on the webpage with the 'Scroll into view' option.
